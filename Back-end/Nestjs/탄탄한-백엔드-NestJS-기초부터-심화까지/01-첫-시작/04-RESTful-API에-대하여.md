# RESTful API에 대하여

# 01 RESTful API란?

클라이언트에서 프론트엔드로 어떤 화면을 보여달라고 요청하면, 프론트엔드 서버는 코드에서 해당하는 UI를 response로 보내 준다.

그러다 API가 필요한 경우가 생기면 백엔드 서버에 request를 하고, 백엔드 서버는 JSON 그리고 그 외의 형태로 응답을 한다.
