# DB intro

### Database

체계화된 데이터의 모임



### Query

- 데이터를 조회하기 위한 명령어
- 조건에 맞는 데이터를 추출하거나 조작하는 명령어
- "Query를 날린다" -> DB를 조작한다



## Database의 기본 구조

### 스키마(schema)

데이터베이스에서 자료의 구조, 표현방법, 관계 등을 정의한 구조



### 테이블(table)

- 열(column): 필드 or 속성
- 행(row): 레코드(record) or 튜플
- 기본키(PK):  각 행(레코드)의 고유값으로 Primary Key로 불린다. 반드시 설정하여야하며, 데이터베이스 관리 및 관계 설정시 주요하게 활용된다.
