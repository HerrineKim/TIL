# HTML이란

## 1. Markup Language

- 태그 등을 이용하여 문서나 데이터의 구조를 명시하는 언어
- 대표적인 예 - HTML, Markdown



## 2. HTML

웹 페이지를 작성(구조화)하기 위한 언어



## 3. HTML 기본 구조

```html
<!DOCTYPE html>
<html lang="ko">
    <head>
        <meta charset="UTF-8">
        <title>Document</title>
    </head>
</html>
<body>
    
</body>
</html>
```

- html: 문서의 최상위(root) 요소

- head: 문서 메타데이터(데이터를 설명해주는 데이터) 요소
  - 문서 제목, 인코딩, 스타일, 외부 파일 로딩 등
  - 일반적으로 브라우저에 나타나지 않는 내용
- body: 문서 본문 요소
  - 실제 화면 구성과 관련된 내용
