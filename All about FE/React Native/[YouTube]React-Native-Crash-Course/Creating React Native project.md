## Expo CLI로 프로젝트 시작하기

https://docs.expo.dev/



1. node.js LTS 버전 설치

2. `npm install -g expo-cli`

3. 프로젝트 생성

   ```bash
   expo init RNCourse
   
   # 'blank' 선택
   
   cd RNCourse
   npm start # you can also use: expo starts
   ```
